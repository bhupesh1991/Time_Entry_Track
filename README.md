# Time Tracking

Time Tracking is for creating daily projects task and track as per start time and end time.

### Prerequisites

require python and Django framework with sqlite database


### Installing

1) Project Repository Clone from bitbucket.
git clone https://github.com/bhupesh1991/Time_Entry_Track.git

2)  Django Installation - django-admin startproject okr


3) Create Virtual Environment -  mkvirtualenv time_env -p python3

4) run command in project folder to install all django packages

-pip install -r requirements.txt

5) Migrate Default Model - python manage.py migrate

6) Create django superuser - python manage.py createsuperuser

12) Run Localhost - python manage.py runserver
